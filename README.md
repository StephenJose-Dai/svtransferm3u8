## 介绍
###### 这是一个将音频、视频转换成m3u8格式并且自动生成m3u8播放地址的小工具，目前支持单个文件上传和批量上传
###### 支持单个转换后的url复制或者多个转换后的url批量导出
## 语言
###### 该工具前端由html、js、css语言编写，后端由python和ffmpeg编写
